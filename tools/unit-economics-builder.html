📐 单位经济核心公式

  • 单笔毛利 = AOV - COGS - 物流 - 平台佣金
  • 年客户收入 = AOV x 复购次数
  • 年客户毛利 = 单笔毛利 x 复购次数 - 广告 - 其他
  • LTV = 年客户毛利 x (1 - 流失率) / 流失率
  • LTV/CAC > 3 = 健康，Payback < 12月 = 安全
