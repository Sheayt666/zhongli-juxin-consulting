📐 分析公式

  • 互动率 = (总点赞 + 总评论) / (粉丝数 × 视频数) × 100%
  • 带货转化率 = 订单数 / (粉丝数 × 直播数) × 100%
  • 粉丝质量评分 = 互动率权重40% + 转化率权重30% + 活跃度权重30%
  • 平均GMV/直播 = 总GMV / 直播数
  • 活跃度 = 视频发布频率 + 直播频率 + 粉丝增长率
